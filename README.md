🌾 About the Project

The AI-Based Farmer Query Support and Advisory System is a web-based application developed to assist farmers by providing instant, AI-powered agricultural guidance. Farmers can enter their name, crop name, location, and farming-related questions through a simple web interface. The application uses Google Gemini AI to generate accurate, easy-to-understand recommendations that help farmers make better decisions regarding crop cultivation and management.

The system is built using Python and the Flask web framework, with SQLite used for storing farmer queries and AI responses. This enables the application to maintain a history of interactions, making it useful for future reference and analysis.

The primary objective of this project is to bridge the gap between farmers and modern agricultural knowledge by offering quick, accessible, and intelligent recommendations. It is designed to be user-friendly so that farmers with minimal technical knowledge can easily use the system.

Key Features
🤖 AI-powered farming recommendations using Google Gemini AI
🌱 Crop-specific agricultural guidance
📍 Location-based query support
💾 Automatic storage of farmer queries and AI responses
💻 Simple and responsive web interface
🗄️ SQLite database integration for data management
⚡ Fast and easy-to-use application
Objectives
To provide instant agricultural advice using Artificial Intelligence.
To improve farmers' decision-making through intelligent recommendations.
To reduce dependency on manual agricultural consultation.
To maintain a database of farmer queries for future reference.
To demonstrate the practical application of AI in the agricultural sector.
Technologies Used
Programming Language: Python
Framework: Flask
Frontend: HTML, CSS, JavaScript
Database: SQLite
AI Model: Google Gemini API
Development Environment: Visual Studio Code
Expected Outcome

The project enables farmers to receive reliable farming advice within seconds. By integrating Artificial Intelligence with a web application, the system offers an efficient, scalable, and user-friendly solution that supports better agricultural practices and promotes the adoption of modern technology in farming.
